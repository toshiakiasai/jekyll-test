# コードテスト

Rubyで書いたコードでkramdownのレンダリングテストをします。

```ruby
class A
  FINAL = "string"      # comment
  validate :symbol, format: /reg[ex]?p/
  array [0, 1_000_000, nil, true, false, ->_{ ; }]

  method do |arg, optional=default, kw: nil, *rest, **kwrest|
  end
end
```

コードブロックの上下はこれくらい空きます。
